### 요약
1. 환경 구간 설정 타당성
- 6개 장소의 평균 데시벨이 40~80dB에 고르게 분포
- 소음 구간 경계(35, 50, 65, 80dB)가 실제 환경을 잘 반영
2. 목표별 음악 속성 차별화
- BPM: 55~114 범위로 명확한 계층 구조
- Energy: 0.20~0.70 범위로 활동-휴식 구분
- Mood: 85~100% 일관성으로 감정 분류 정합성 확보
3. dB-음악 상관관계
- 전체적으로는 약한 상관이지만, Active/Focus/Reading 목표에서 명확한 마스킹 효과 확인
- 66-80dB 구간에서 마스킹 효과 최대화
- Meditate는 반대 전략(소음↑ → BPM↓)으로 차별화
4. 규칙 엔진 완전성
- 전체 42개 조합 100% 커버
- 이상치, 누락, Fallback 0건
- 장소×목표별 일관된 패턴 유지
5. 음악 속성 일관성
- BPM ↔ Energy 상관계수 0.977로 물리적-심리적 속성 완벽 일치
- 변수 간 독립성 확보로 안정적 규칙 엔진 구조

### 6. 음악 속성 타당성 검증

6.1 BPM vs Energy 상관관계

상관계수: r = 0.977 (p < 0.001)

| 목표 (Goal) |   평균 BPM   | 평균 Energy |  상태 | 해석    |
| :-------- | :--------: | :-------: | :-: | :---- |
| Active    | **114.17** |  **0.70** |  ✅  | 높음–높음 |
| Focus     |    80.29   |    0.45   |  ✅  | 중간–중간 |
| Relax     |    66.92   |    0.37   |  ✅  | 낮음–낮음 |
| Sleep     |    55.00   |    0.20   |  ✅  | 최저–최저 |

>"빠른 곡 = 강한 에너지, 느린 곡 = 부드러운 에너지" 원칙 일관성 확보
>물리적 속도와 심리적 강도의 완벽한 일치

>Decibel은 1차 결정 변수가 아닌 2차 보정 변수로 활용
>Goal이 음악 선택의 중심, Decibel은 강도 조정의 슬라이더 역할
>변수 간 과도한 중복 없어 다중공선성 문제 없음 (향후 ML 전환 용이)
