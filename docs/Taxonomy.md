#  Soundscape — Taxonomy v1  
> 위치(Location) × 데시벨 구간(Decibel Band) × 목표(Goal)를 기반으로 음악 속성 규칙을 설계하기 위한 분류 체계

---

##  1. 위치(Location)
| 분류 | 세부 예시 | 설명 |
|------|------------|------|
| `cafe` | 스타벅스, 투썸 등 | 주변 대화와 배경음이 섞인 중간 소음 환경 |
| `library` | 독서실, 공부방 | 정적·집중 환경 |
| `co-working` | 사무실, 코워킹 스페이스 | 일정한 백색소음, 업무 중심 환경 |
| `moving` | 이동중 | 간헐적 진동음·잡음이 있는 이동 환경 |
| `gym` | 헬스장 | 반복적인 기구 소음과 음악이 섞인 활기찬 운동 환경 |
| `home` | 거실, 방, 베란다 | 개인화된 편안한 환경 |
| `park` | 공원, 산책로, 야외 벤치 | 자연의 소리와 사람들의 대화가 섞인 개방적 휴식 환경 |

---

##  2. 데시벨 구간(Decibel Band)
| 구간 | dB 범위 | 환경 예시 | 분류명 |
|------|----------|------------|--------|
| `silent` | 0–35dB | 독서실, 새벽 주택가 | 정적 |
| `quiet` | 36–50dB | 사무실, 도서관 | 조용 |
| `moderate` | 51–65dB | 카페, 식당 | 보통 |
| `loud` | 66–80dB | 거리, 지하철 내부 | 시끄러움 |
| `very_loud` | 81-100dB  | 공연장, 공사장 | 매우 시끄러움 |

---

##  3. 목표(Goal)
| Goal | 설명 | 대표 상황 |
|------|------|------------|
| `focus(집중)` | 집중을 유지하거나 업무 효율을 높이기 위한 상태 | 공부, 업무, 코딩 |
| `relax(휴식)` | 휴식과 안정, 스트레스 완화 | 카페, 퇴근 후 |
| `sleep(수면)` | 수면 유도 또는 취침 전 안정 | 야간, 숙면 모드 |
| `active(활력)` | 활동적 에너지, 운동/이동 중 사용 | 운동, 출근길 |
| `anger(분노)` | 강한 감정이나 스트레스를 해소하고 싶은 상태 | 감정 폭발 직후, 분노나 스트레스 해소가 필요한 상황 |
| `consolation(위로)` | 정서적 안정과 감정 회복을 필요로 하는 상태 | 위로가 필요한 순간, 감정 조절이 필요한 상황 |
| `neutral(미설정)` | 목표 미설정 상태 | 일상생활 |
